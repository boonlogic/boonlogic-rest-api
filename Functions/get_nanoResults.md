# **GET nanoResults**
<br/>

#### Returns the specified clustering results
##### input
>`instanceID`
>>*label for the nano*
>
>`results`
>>*options for results to return relating to the most recent patterns clustered*  
>>*NOTE: see [Guide: Nano Results](../Guides/Guide_Nano_Results.md) for more information*

#### output
```json
  {
    "ID": [
      0
    ],
    "SI": [
      0
    ],
    "RI": [
      0
    ],
    "DI": [
      0
    ],
    "MD": [
      "string"
    ]
  }
```
